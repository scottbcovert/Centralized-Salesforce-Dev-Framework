  getTriggerDispatcher

private static ITriggerDispatcher getTriggerDispatcher(Schema.sObjectType soType)
DescriptionGets the appropriate dispatcher based on the SObject. It constructs the instance of the dispatcher dynamically using the Type API. The name of the dispatcher has to follow this format: TriggerDispatcher. For e.g. for the Account object, the dispatcher has to be named as AccountTriggerDispatcher. For the custom object Custom__c, the name should be CustomTriggerDispatcher.
AuthorScott Covert
Date11/9/2014
ReturnsITriggerDispatcher A trigger dispatcher if one exists, otherwise null.
